# 欢迎来到 Git Game（中文版）

这是一个**Shell终端游戏**，旨在测试你对 Git 命令的了解。游戏中的每个级别都是在此仓库上执行的任务。完成任务后，你将获得下一个任务。初级总共有十个级别，每个级别比上一个更难！

后续会开放中级仓库～并欢迎大家提供关卡。
